## 実行までの流れ  
`$ cd ~/catkin_ws/src`  
`$ git clone git@github.com:ryosukehayashi3/mypkg2.git`  
`$ cd ..`  
`$ catkin_make` 

`$ roscore`  
`$ rosrun mypkg2 count.py`  
`$ rosrun mypkg2 twice.py`  
`$ rostopic echo /twice`
